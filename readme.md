# sequencer
**sequencer** is a blockchain built using Cosmos SDK and Tendermint and created with [Ignite CLI](https://ignite.com/cli).

## Get started

```
make install
go mod tidy
```

## Quick Start
```
# start one node
sh node0.sh

# start mulit nodes
sh multi-node.sh
```
